Exercicio práctico: edición dixital de Contos da miña terra

  1. Abre Visual Studio Code. Preme Ctrl + Shift + P e escribe no caixa de texto: dev containers. Selecciona a opción Dev containers: Open Folder in Container...
  2. Selecciona o cartafol que descargaches no punto 2.1
  3. Cando remate de cargar, vai ao enderezo http://localhost:8080localhost:8080/exist/apps/dashboard
  4. Prepara un documento de Microsoft Word que conteña o texto de Contos da miña terra tirado de Wikisource
  5. Engade estilos para anotar o documento, seguindo a convención tei:nome-elemento. Isto xerará elementos TEI con ese nome ao importar o arquivo. Por exemplo, para crear unha edición paralela texto-facsímile necesitamos marcar o inicio de cada páxina. Crea o estilo tei:pb e vai engadindo <n="01"> nese estilo cada vez que comece unha nova páxina (en 3.3 descargaches unha carpeta con todas as páxinas do facsímile). Durante a importación do documento, esa anotación converterase nun elemento <pb n="01"/> (cf. https://tei-c.org/release/doc/tei-p5-doc/en/html/ref-pb.html). Utilizaremos o valor do atributo @n para facer a referencia ás imaxes.
  6. Seguindo as instrucións fornecidas durante a sesión, xerarás unha aplicación web usando o xerador de apps de TEI Publisher.
  7. Utilizarás o editor eXide para editar o modelo HTML da túa edición (ex. templates/pages/facsimile.html). Poderás engadir funcionalidades á túa edición con facilidade grazas aos compoñentes web de TEI Publisher
  8. Utilizrarás o editor de ODD para engadir instrucións que procesen o XML co texto da edición: localhost:8080/exist/apps/NOME_DA_APP/odd-editor.html?odd=NOME_ODD.odd
